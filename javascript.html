JavaScript

1. 介绍引入JavaScript

一切数据都由客户端游览器发出, 将请求发送到服务器去做校验, 用户完成注册功能, 但是由于需要数据校验花费的时间很长, 长达10分钟

特点:

  1. JavaScript是运行在游览器端的, 阻断式解释执行的前端脚本语言
  2. 弱类型语言: 声明变量时不需要指定变量类型
  3. JavaScript的主要作用: 做前端的数据校验, 用户交互
  4. 基于对象(与Java面向对象的思维很像) 和 事件驱动的语言
  5. 动态语言: 程序在运行期间, 仍然可以动态改变程序结构
  6. JavaScript=ECMAScript 语法 + BOM游览器对象模型 + DOM文档对象模型


2. JavaScript入门案例

  1. JavaScript写在哪: head里 或者 body下
  2. 如果想让JavaScript代码受控, 应该写在方法里, 然后标签里的事件进行调用
  3. 一个方法可以被多个事件调用, 一个事件可以调用多个方法
  4. 一对script标签只做一件事: 引入外部文件: 内部代码编写





3. ECMAScript语法

(1). 弹框

输出: 弹框输出 alert("我是方法3");
输入: 弹框输入 prompt("输入你的信息: ")

(2). 变量

变量: 先声明 赋值 使用
  • const age=20 创建一个常量
  • let age=20 创建一个局部变量
  • var age=20 创建一个成员变量
    • [扩展一个隐患问题]: 如果一个变量不加关键字限制, 一旦该变量被使用过一次就会由局部变量变成全局变量
    变量类型: number, string, boolean

    4. JavaScript运算符

    • 算数运算符: + - * / % ++ --
    • 逻辑运算符: && || ! 也是断路运算符

    扩展: = == 和 === 的区别
    1. = 是赋值运算符
    2. == 是先进行类型判断, 再进行值的判断
    3. === 是直接进行值的判断
